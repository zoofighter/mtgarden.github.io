<h1 id="&#49707;&#51088;&#54632;&#49688;">&#49707;&#51088;&#54632;&#49688;<a class="anchor-link" href="#&#49707;&#51088;&#54632;&#49688;">&#182;</a></h1><ul>
<li>Pandas는 데이터 컬럼 내에(데이터 개별 요소) 숫자함수를 적용할 수도 있고  컬럼(columns)간, 로우(row)간에도 숫자함수를 적용할 수 있다. </li>
<li><p>자주 사용되는 숫자함수는 다음과 같다.</p>
<ul>
<li>+, -. *, / :  같은 사치역산</li>
<li>mod :      나머지 구하기(%)</li>
<li>abs :      절대값  </li>
<li>round :    소수점 특정 자릿수에서 반올림하기.</li>
</ul>
</li>
<li><p>Pandas는 numpy의 범용함수를 지원한다.</p>
<ul>
<li>np.power  </li>
<li>np.sqauare </li>
<li>np.log</li>
<li>np.exp   </li>
<li>np.sign</li>
</ul>
</li>
</ul>
